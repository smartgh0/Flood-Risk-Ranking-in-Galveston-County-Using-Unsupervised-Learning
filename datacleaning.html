Data Cleaning

Data Wrangling:

Initial Dataset: 21 columns and 516 observations, combining geographical, environmental, and socio-economic factors.

Cleaning: Removed missing values, standardized formats, and filtered irrelevant data to ensure accuracy and consistency.

Feature Engineering: Created new features from existing data to enhance model performance, including risk indices and proximity scores.

Normalization: Applied MinMaxScaler to normalize data, ensuring all features were on a similar scale for clustering analysis.

  • TotalZestimate was dropped because it had very high values as compared to other features.
